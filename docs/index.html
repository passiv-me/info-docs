InfoDocs - Informatica per principianti¶

Benvenuto su InfoDocs, lezioni di informatica per tutti.

Questo sito raccoglie una serie di lezioni di informatica per principianti. Nessuna esperienza richiesta!

DOMANDA: Perchè questo sito? Internet è già pieno di risorse utili per imparare l’informatica.

RISPOSTA: Verissimo, purtroppo queste risorse sono spesso in inglese, e questo impedisce a molte persone di avvicinarsi alla materia.

Lo scopo di questo sito è di essere una risorsa completa, 100% in italiano e pensata per principianti, per imparare l’informatica e la programmazione.

Conoscere l’informatica è una dote importante da avere nel 21esimo secolo. L’informatica ha cambiato e continua a cambiare la società in cui viviamo. Conoscere l’informatica ci aiuta a capire e conoscere il mondo.

Inoltre, imparare l’informatica è un ottimo investimento del proprio tempo. Non solo l’industria informatica è in pieno BOOM e crea nuovi posti di lavoro ogni giorno, ma è ormai difficile pensare ad un lavoro che non includa l’avere a che fare con un computer/tablet o smartphone. Imparare a programmare apre un mondo di opportunità!

Bene… se siete arrivati fino a qui, COMPLIMENTI!

Siete abbastanza motivati per procedere alla prima lezione, dove scopriremo «cos’è un computer?».

WORK IN PROGRESS¶

File - un file è un insieme di bytes memorizzato nella memoria persistente del computer (disco rigido o hard disk).

Un file è utile se i bytes al suo interno rappresentano qualche informazione. Ad esempio, un file può contenere un’immagine, un video, il testo della divina commedia, etc.

Dato che un file è semplicemente un insieme di bytes, come fa a contenere una varietà così grande di informazioni?

Semplice! Il contenuto di un file viene interpretato in modo diverso dal computer in base al formato del file.

Avrete certamente sentito parlare di formati «famosi» come .pdf (per documenti stampabili), .mp3 (per file musicali), .doc (per documenti Word), .ppt (per PowerPoint), etc.

I formati sono semplicemente un insieme di regole che determinano come un file va scritto/letto dal computer in modo da essere interpretato correttamente.

Ad esempio, un applicazione che genera files in formato .mp3 deve sapere come scrivere un file in modo che venga letto correttamente da un’applicazione che legge .mp3 (ad esempio Spotify).

Se le regole del formato non vengono rispettate, possono sorgere errori come «Contenuto non valido», o «File corrotto». Per esempio provate ad aprire un file .mp3 con Word e vedere cosa succede.

Riassumendo:

  • un file è un insieme di bytes salvato su memoria persistente.

  • il modo in cui vengono interpretati i bytes del file dipende dal formato del file.

  • i files (e i formati) sono molto utili per scambiare informazioni tra applicazioni diverse.
